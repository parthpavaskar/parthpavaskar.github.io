# Personal Academic Website

This is my personal academic website showcasing my research, publications, talks, and professional experience in computational/theoretical plasma astrophysics.

### Live Site

Visit the live site [here](https://parthpavaskar.github.io/)

### Build

This website is built using:

- **[Hugo](https://gohugo.io/)** - Static site generator
- **[Hugo Blox](https://hugoblox.com/)** - "Academic CV" template used as the base framework
- **[Tailwind CSS v4](https://tailwindcss.com/)** - CSS framework
- **[Pagefind](https://pagefind.app/)** - Static search library
- **[Preact](https://preactjs.com/)** - React alternative for interactive components

### Customizations

This site extends the Hugo Blox Academic CV template with:

- Custom CSS styling for publication views and landing pages (`assets/css/custom.css`, *AI assisted*)
- Modified resume blocks (experience, skills, awards, contributions)
- Custom text/element spacing
- Custom width controls for content sections
